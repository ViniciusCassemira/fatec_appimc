# App IMC

Projeto acadêmico desenvolvido no Android Studio utilizando Java,
como parte de estudos da matéria Programação para Dispositivos Móveis. O aplicativo se baseia em coletar o peso e a altura do usuário, calculando assim o seu IMC (Índice de Massa Corporal) e trazendo os resultados na sequência.
